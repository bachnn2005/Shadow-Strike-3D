## Introduction

**Counter Strike: Boss Raid** is a 3D third-person shooter game developed using Unity.
The game focuses on fast-paced shooting mechanics where the player fights waves of enemies and finally faces a powerful boss.

This project was created as a learning project to practice game development concepts such as player control, shooting systems, enemy AI, animations, and UI systems.

---

## Features

### Player System

* Third-person player movement
* Aiming system
* Health and damage system
* Gravity and character controller movement

### Gun System

* Shooting mechanics
* Reload system
* Gun sound effects
* Weapon animation support

### Enemy System

* Enemy AI movement
* Enemy attack system
* Enemy health and damage

### Boss System

* Boss health system
* Boss battle mechanics
* Win screen when boss is defeated

### UI System

* Boss health bar
* Win panel
* Game statistics display

---

## Technologies Used

* **Game Engine:** Unity
* **Programming Language:** C#
* **Audio System:** Unity AudioSource
* **Animation:** Unity Animator & IK system
* **UI:** Unity Canvas
